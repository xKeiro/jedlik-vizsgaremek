# Project guides

## How to run the project

- Download and install docker
- Run Docker Desktop
- Download the project
- Open a Terminal
- Write the following commands:
  ```
  cd '/path/to/project'
  docker compose up -d
  ```
  - The application is available on your [localhost](http://localhost)
